# Contact Manager

A console-based Contact Manager application built using Core Java.

## Features
- Add Contact
- View Contacts
- Update Contact
- Delete Contact
- Search Contact

## Concepts Used
- Object-Oriented Programming
- Collections Framework
- Interfaces
- Exception Handling
- Layered Design

## Project Structure
src/
├── model/
├── service/
└── Main.java

## Future Enhancements

- Store contacts in a file
- Store contacts in a database
- Add contact validation
- Build REST APIs using Spring Boot
